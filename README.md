# 🧮 Simple Calculator

A clean and responsive calculator web app built using HTML, CSS, and JavaScript.  
This project performs basic arithmetic operations with a modern UI design.

---

## 🚀 Live Demo
👉 https://simple-calculator-delta-nine.vercel.app

---

## ✨ Features
- ➕ Addition, ➖ Subtraction, ✖ Multiplication, ➗ Division  
- ⚡ Instant calculation  
- 🎯 Clean and minimal UI  
- 📱 Responsive design (works on mobile & desktop)  
- 🎨 Gradient background and styled buttons  

---

## 🛠️ Tech Stack
- HTML5  
- CSS3  
- JavaScript (Vanilla JS)
